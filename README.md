# IFIP WMNC 2024 — Conference Website Archive

This repository contains a complete static archive of the official website of the **15th IFIP Wireless and Mobile Networking Conference (IFIP WMNC 2024)**, held at Ca' Foscari University of Venice, Italy, on **11–12 November 2024**.

The archive is maintained to ensure long-term public accessibility of the conference information, as the original website hosted at Ca' Foscari University is no longer available.

## About the Conference

IFIP WMNC 2024 was a discussion forum for researchers, professionals, and students working on novel developments in mobile and wireless networks, services, applications, and mobile computing. The conference combines three historic IFIP events:

- **PWC** — Personal Wireless Communications
- **MWCN** — Mobile and Wireless Communication Networks
- **WSAN** — Wireless Sensors and Actor Networks

The event was technically sponsored by **IFIP** (International Federation for Information Processing) and organized by Ca' Foscari University of Venice.

## Contents

The archive includes:

- Complete conference homepage (programme, committees, venue, call for papers, contacts, photo gallery)
- PDFs published on the official site (Final Program, Publication Ethics Statement)
- All static assets needed to render the site correctly offline:
  - CSS stylesheets (Bootstrap 5, TYPO3 theme)
  - JavaScript bundles
  - Web fonts (Open Sans, Roboto, Bootstrap Icons)
  - Images and logos

All asset URLs have been rewritten to relative paths so the site renders correctly without any external dependencies.
